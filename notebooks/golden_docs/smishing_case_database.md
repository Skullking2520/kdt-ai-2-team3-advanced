[사례 1] 공공기관 사칭 + 과태료 결제 유도
원문: "[교통범칙금 안내] 귀하의 차량(12가3456)이 20XX-03-15 속도위반으로
적발되었습니다. 미납 시 면허 정지 조치됩니다. 납부: http://fine-pay[.]top"
분석:
- 차량번호 삽입 → 표적형 개인화로 신뢰도 상승
- 미납 시 불이익 → 위협-구제 프레임
- 변조 도메인(fine-pay[.]top) → 합법성 신호 모방

[사례 2] 건강보험 환급 사기
원문: "[국민건강보험공단] 2024년 건강보험료 과납분 87,600원 환급 대상입니다.
7일 내 신청하지 않으면 소멸됩니다. 신청: http://nhis-refund[.]net/req"
분석:
- 소액 환급 → 가치 있는 정보처럼 보이나 의심 임계값 하회
- 7일 소멸 → 인위적 마감기한
- 실제 공단 도메인(nhis.or.kr)과 유사한 피싱 도메인

[사례 3] 가족 사칭 긴급 자금 요청
원문: "엄마 나 핸드폰 고장났어. 친구 폰 빌렸어. 급하게 30만원만 보내줘.
계좌: 신한 110-XXX-XXXXX (홍길동). 나중에 꼭 갚을게"
분석:
- 가족 관계 사칭 → 정서적 유대 악용(감성적 사회공학)
- 기기 고장 명목 → 검증 불가 조건 설정
- URL 미포함 → 기존 URL 필터 우회

[사례 4] 앱 업데이트 위장 악성코드 배포
원문: "[카카오뱅크] 보안 강화를 위해 최신 버전 업데이트가 필요합니다.

분석:
- 악성 APK 다운로드 유도 → 원격 제어, 금융정보 탈취
- 실제 앱 스토어 경유를 우회하는 사이드로딩(Sideloading) 공격

[사례 5] 정부 지원금 사기
원문: "[고용노동부] 청년 취업 지원금 300만원 수령 대상으로 선정되었습니다.
신청 마감: 2025-04-30. 본인 확인 후 즉시 지급: http://moel-youth[.]kr"
분석:
- 고액 지원금 → 강력한 금전적 유인(financial incentive)
- 마감일 설정 → 시간 제한 압박
- 실제 고용노동부(moel.go.kr)와 유사 도메인

[사례 6] 음식물 쓰레기·환경 행정 위반 사칭
원문: "음식물 미분리수거 신고가 접수되었습니다. hxxps://조회바람[.]top"
원문2: "[환경관리법 제68조위반] 음식물 혼합투기 내용 메시지를 보내드립니다. hxxps://고지서[.]cc"
원문3: "[통보서발급안내] 제32조3항 혼합투기 사실확인내용 전송하였습니다. hxxps://yor[.]uno"
원문4: "행정관리청 제12조3항 위반 행정처분 안내서 전송완료 hxxps://moy[.]pw"
분석:
- 법 조항 번호 명시(제68조, 제32조3항) → 공문서 위장
- 죄책감 유발 → 수신자가 실제 위반했을 가능성 자극
- [국제발신] 태그 → 행정 시스템 발신처럼 위장
- 변형 도메인(.top, .cc, .uno, .pw) → 비공식 TLD 사용
- 실제 환경부·지자체는 SMS로 URL을 전송하지 않음

[사례 9] 고유가 피해지원금 사칭
원문: "고유가 피해지원금 정부지원금 신청 시작! 개인정보를 입력해주세요. 신청 접수: [URL]"
원문2: "[고유가 피해지원금 신청 안내] 귀하는 고유가 피해지원금 신청 대상자에 해당됩니다."
원문3: "고유가 피해지원금 신청이 접수되었습니다. 다시 한번 확인 부탁드립니다. (https://urrl.kr/25yp3q)"
분석:
- 실제 정부 고유가 대책 발표 직후 집중 발송 → 정책 시의성 악용
- 신청 안내 → 신청 접수 완료 순서로 연속 발송 → 신뢰도 상승
- 개인정보 입력 직접 요구 → 정보 탈취 목적 노골화
- 단축 URL(urrl.kr) → 실제 URL 숨김

[사례 10] OTT 개인정보 유출 사칭
원문: "[Web발신] 최근 티빙 개인정보 유출 사태 관련하여 피해 구제 조회 안내드립니다."
원문2: "[TIVING] OOO 고객님의 소중한 개인정보 유출 정황이 확인되어 안내드립니다."
분석:
- 실제 발생한 개인정보 유출 사고에 편승 → 언론 보도 활용
- TIVING(오기) vs TVING(정식) → 브랜드 오기재로 식별 가능
- 피해 구제 명목으로 더 많은 개인정보 요구

[사례 11] 수사기관·법원 사칭
원문: "검찰청 사이버수사부 김현성 수사관입니다. 당신의 계좌가 범죄에 연루됐습니다."
원문2: "[Web발신][최종통보] KG모빌님 법원소송예정 21000원 8104-4-** 우체국"
분석:
- 수사관 실명(김현성) 명시 → 공식 연락처럼 위장
- 계좌 범죄 연루 → 극도의 공포 유발
- 소액(21,000원) + 우체국 → 의심 임계값 낮춤
- 실제 수사기관은 SMS로 수사 내용을 통보하지 않음

[사례 12] 전자결제 대행사 피해보상 사칭
원문: "저희는 국내 전자결제 대행사 XXX페이입니다. 최근 해킹사태와 관련하여
피해보상 절차 안내를 위해 문자를 드립니다. 피해보상금 7,781,208원,
피해보상 공지확인 https://dxxx.xx, 텔레그램 접수 https://xxx.xxx"
분석:
- 7,781,208원 → 불규칙한 고액으로 실제 피해 산정처럼 위장
- 텔레그램으로만 접수 → 추적 회피 목적
- "개인정보 보호를 위해 텔레그램으로만" → 역설적 논리 사용

공통 탐지 지표 요약:
- 변형 도메인: .top, .ink, .xyz, .pw, .uno 등 비공식 TLD
- 긴급성 표현: "즉시", "미진행 시 차단", "기한 내", "최종통보"
- 직접 정보 요구: URL 없이 계좌번호·카드번호 직접 삽입
- 법 조항 번호 명시: "제68조", "제32조3항" 등
- 텔레그램 유도: 공식 채널 대신 텔레그램으로 접수 유도
- 구체적 금액: 불규칙한 금액으로 실제 산정처럼 위장
- 차량번호·이름 개인화: 수집된 개인정보로 표적형 신뢰 형성
